# QA Pretrainee - Практические задания

Это репозиторий с практическими заданиями по тестированию. Здесь собраны материалы, демонстрирующие навыки составления тестовой документации, работы с дефектами и анализа требований.

## О проекте

Этот репозиторий содержит результаты практических работ, выполненных в рамках обучения QA-тестированию. Каждый проект включает полный цикл тестовой документации: от анализа требований до составления отчётов о тестировании.

##  Структура репозитория

###  Тестирование Youtube
Комплексное тестирование видеоплатформы YouTube. Включает:

- **Модули, Чек-листы, Тест-сьюиты, Тест-кейсы.pdf** - Декомпозиция функционала YouTube на модули, создание чек-листов проверок с приоритизацией (Smoke, Acceptance Testing, Manual Acceptance Testing), формирование тест-сьютов и детальных тест-кейсов.
- **Описание дефектов.pdf** - Примеры оформленных баг-репортов с указанием шагов воспроизведения, ожидаемых и фактических результатов, серьёзности (Severity) и приоритета (Priority) дефектов.
- **Определение видов тестирования.pdf** - Практика по классификации тестирования: позитивное/негативное, функциональное/нефункциональное на реальных сценариях YouTube.
- **Пользовательские требования, истории, сценарии и Юз-кейсы.pdf** - Сбор и анализ пользовательских требований, написание User Stories и Use Cases для функционала платформы.
- **Тестирование формы ввода по чек-листу.pdf** - Практическая проверка полей ввода с использованием чек-листа.

### Тестирование игры
Тестирование игрового приложения "Made in abyss" - дипломного проекта студента прошлых лет. Включает:

- **Диаграмма Гантта.pdf** — распределение задач между разными отделами команды, занимающимися функциональным, нагрузочным и другим тестированием в течение 3 недель.
- **Тест-кейсы по модулям игры.pdf** — Написание тест-кейсов для проверки ключевых модулей игры с учётом игровой логики и пользовательских сценариев.
- **Отчёт о тестировании - Презентация.pdf** — Итоговая презентация с результатами тестирования, статистикой обнаруженных дефектов и рекомендациями по качеству продукта.
